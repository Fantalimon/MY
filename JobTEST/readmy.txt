В файле config.ini находятся константы изменяя их на свои значения в кавычках можно успешно пользоваться скриптом отправки в базу данных о юзере. Естественно базу-данных нужно перед этим создать.

Для создания базы данных воспользуйтесь файлом
comand _for_greate database.txt

в нем после каждой запятой нажмите ctrl+Enter пошагово с верху вниз.
в phpStorm это не сложно.

создайте базу данных с именем
jobtest

выполните команду
mysql -u root -p jobtest < /полный путь к файлу (backupjobtest.sql)который видно в репозитории/backupjobtest.sql
