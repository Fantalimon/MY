В файле config.ini находятся константы изменяя их на свои значения в кавычках можно успешно пользоваться скриптом отправки в базу данных о юзере. Естественно базу-данных нужно перед этим создать.

form - переход на главную(страницу формы)

// Почта администрантору

adminemail - адресс почты администратора
mailthem - тема письма

// коннект к базе данных

user - имя пользователя базы данных
password - пароль от базы данных
dbname - имя базы данных
tbname - имя таблици в базе данных
port  - порт ввода вывода
charset - кодировка по умолчанию

// вывод записей на странице

pagainlist - эта константа отвечает за колличество строк отображаеммых на листе в админ панели.

сlass SendMail - отвечает за отправку формы
